# Builder.io Design System

Welcome to the **Builder.io Design System** - a comprehensive, accessible, and modern component library built with **Tailwind CSS** and optimized for **Builder.io** integration.

## üéØ Purpose

This design system provides a consistent foundation for building user interfaces that are:

- **Accessible**: WCAG 2.1 AA compliant components
- **Responsive**: Mobile-first design patterns
- **Consistent**: Unified visual language across all components
- **Developer-friendly**: TypeScript support with comprehensive documentation
- **Builder.io optimized**: Seamless integration with Builder.io's visual editor

## üß© Components

### Button Component

Our flagship component with comprehensive variants and states:

- **4 Variants**: Primary, Secondary, Outline, Ghost
- **3 Sizes**: Small, Medium, Large
- **Multiple States**: Default, Hover, Active, Disabled, Loading
- **Advanced Features**: Icons, Links, Full-width, Builder.io compatibility

## üé® Design Tokens

### Color System

Built on Tailwind CSS principles with semantic color naming:

- **Primary Colors**: Blue scale (50-950) for main actions
- **Secondary Colors**: Gray scale (50-950) for neutral elements
- **Semantic Colors**: Success, Error, Warning for status indication

### Typography Scale

Consistent text sizing based on a modular scale:

- **xs**: 12px - Fine print and captions
- **sm**: 14px - Secondary text
- **base**: 16px - Body text (default)
- **lg**: 18px - Emphasized text
- **xl**: 20px - Small headings

### Spacing Scale

4px-based spacing system for consistent layouts:

- Padding: `p-1` (4px) to `p-24` (96px)
- Margins: `m-1` (4px) to `m-24` (96px)
- Gaps: `gap-1` (4px) to `gap-24` (96px)

## üöÄ Getting Started

### Installation

```bash
npm install builderio-ds
```

### Basic Usage

```jsx
import { Button } from "builderio-ds";

function App() {
  return (
    <Button variant="primary" size="medium">
      Get Started
    </Button>
  );
}
```

### With Tailwind CSS

Import the CSS for styling:

```jsx
import "builderio-ds/dist/styles.css";
```

### Builder.io Usage

```jsx
// Register components
import { registerBuilderComponents } from "builderio-ds";
await registerBuilderComponents();

// Use in Builder.io
<Button text="Click me" variant="primary" />;
```
